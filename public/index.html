Add medias from list to play them in a continuous mix


This is the web version of a mobile cross-platform app for multimedia mixing. The application is a student project developed for computing studies at Université du Québec à Trois-Rivières. The repository is public on GitHub in swift (with CoreData) for iOS and java (with SQL) for Android. Both mobile devices can synchronize data through Google's Firebase.
